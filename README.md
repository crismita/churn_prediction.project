# churn_prediction.project
Customer churn prediction focuses on identifying customers who are likely to stop using company's services and it is done with the help of machine learning.

This project predicts whether a customer is likely to churn based on key customer attributes. The goal is to help businesses identify at-risk customers early and take proactive steps to improve customer retention.


---

## Objective

Predict customer churn using machine learning

Analyze important factors influencing churn

Provide an easy-to-use prediction interface using Streamlit



---

## Machine Learning Approach

Data preprocessing and feature encoding

Feature selection including gender, tenure, and monthly charges

Trained a classification model using Scikit-learn

Model used to generate churn predictions



---

## Dataset Description

The dataset includes the following features:

Gender: Customer gender (encoded)

Tenure: Duration of customer relationship

Monthly Charges: Monthly billing amount

Churn: Target variable indicating whether the customer churned
